# Terminal Craft ⛏️

Python ile geliştirilmiş, terminal tabanlı minimalist bir Minecraft klonu ve RPG prototipi.

## 🚀 Özellikler
* **Karakter Seçimi:** Steve veya Alex karakterleriyle oyuna başlama.
* **Kayıt Sistemi:** JSON tabanlı otomatik veri kaydetme ve yükleme sistemi.
* **Madencilik (Mining):** Blokların dayanıklılık (durability) seviyelerine göre kazı yapma ve envantere ekleme.
* **ASCII Art:** Terminalde görselleştirmeler.
